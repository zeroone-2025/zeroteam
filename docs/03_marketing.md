| 장 | 제목 | 본문 카피 | 시각 요소 |

| 2장 (공감) | "주요 기능 소개" | 1. 딴 노래 나옴 2. 배속 엉망 3. 아예 안 나옴 (진지) | 기능 소개인데 전부 단점만 나열, 진지한 표정 아이콘 |

| 4장 (CTA) | "지금 바로 체험하세요" | 안 하셔도 됩니다. 근데 궁금하잖아요. | QR코드 옆에 "누르지 마세요" |

### 2. 에브리타임 글

- **목적**: 대학생 커뮤니티에서 자연스러운 경험 공유형 추천
- **포맷**: 제목 + 본문 3-5줄 + 링크
- **작성 가이드**:
  - 광고 느낌 없이, "이런 거 써봤는데 괜찮아서 공유" 톤
  - 대학별 특성에 맞게 커스텀 (예: 과제, 팀플, 시험 등 연결)
  - 댓글 대응 준비 (예상 질문과 답변)

#### 버전 1 (확정)
- **제목**: 이 앱 쓰지 마세요 진심으로
- **본문**: 노래 틀었는데 안 나옴. 다시 틀었는데 다른 노래 나옴. 또 틀었는데 2배속으로 나옴. 진짜 화나는데 왜 자꾸 틀게 되냐ㅋㅋㅋㅋ 절대 쓰지 마세요 시간 낭비임 [링크]
- **상황**: 스트레스 해소 / 역홍보

#### 버전 2
- **제목**: 팀플하다가 발견한 역대급 쓸데없는 사이트
- **본문**: 팀플 회의 중에 팀원이 공유해줬는데 노래 틀면 딴 노래가 나오거나 0.5배속으로 나오거나 아예 안 나옴ㅋㅋㅋ 회의 분위기 환기용으로 최고임. 다 같이 폰으로 틀어보고 누가 묵음 당하나 내기함 [링크]
- **상황**: 팀플 / 모임 분위기 전환

#### 버전 3
- **제목**: 스트레스 받을 때 이거 해보셈
- **본문**: 시험공부 하다가 멘탈 나갈 때 이거 켜서 노래 틀어봐. 내가 고른 노래 무시하고 딴 거 틀어주는 앱인데 화나는 걸 넘어서 웃김ㅋㅋ 친구들한테 보내면 다들 "이게 뭐야" 하면서 3번은 더 누름 [링크]
- **상황**: 시험 기간 / 스트레스 해소

> **AI 프롬프트** -- 에브리타임 추천글 생성
>
> ```
> @docs/01_plan.md 참고해서 에브리타임 추천글 3개 버전을 만들어줘.

> 광고 느낌 없이 실제 사용 경험 공유 톤으로,
> 각 버전은 다른 상황(과제, 팀플, 일상)에서 출발하고 링크 포함해줘.
> ```

- **체크리스트**:
  - [ ] 광고 느낌 없는 자연스러운 톤 확인

### 5. 블로그 포스트

- **목적**: 제품 아카이빙 및 SEO를 통한 검색 유입
- **포맷**: 네이버 블로그 1편 (이미지 5장 이상)
- **작성 가이드**:
  - 제목에 검색 키워드 포함 (예: "대학생 팀플 관리 앱")
  - 구성: 문제 제기 → 솔루션 소개 → 주요 기능 시연 캡처 → 사용 후기 → 링크
  - 카드뉴스 이미지와 시연 캡처를 본문에 활용

- **제목**: 노래 틀면 딴 노래 나오는 앱 만들어봤습니다 (쓸데없는 앱 리뷰)
- **본문 구성**:
  1. 문제 제기: "여러분, 음악 앱이 너무 친절하다고 느낀 적 없으신가요? 내가 고른 노래를 그대로 틀어주다니... 너무 성의 없잖아요."
  2. 솔루션 소개: "그래서 만들었습니다. 딴짓뮤직. 당신이 고른 노래를 절대 그대로 틀어주지 않는 세상에서 가장 쓸데없는 뮤직 플레이어."
  3. 기능별 시연 캡처: 딴 노래 재생 / 랜덤 배속 / 묵음 각각의 시연 캡처 + "이게 버그가 아니라 기능입니다"
  4. 사용 후기: "별점 1점 리뷰가 곧 성공의 증거입니다" + 팀원 리액션 사진
  5. CTA: "쓸데없는 경험, 지금 시작하세요 (안 해도 됩니다)" + 링크
- **해시태그**: #딴짓뮤직 #쓸데없는앱 #제멋대로재생 #묵음챌린지 #랜덤배속 #B급감성 #스트레스해소 #웃긴앱 #대학생추천 #해커톤

## 바이럴 전략

### 역홍보 컨셉
I'M KING의 핵심 마케팅 전략은 **"쓰지 말라고 하면서 쓰게 만드는"** 역발상 홍보입니다.

- **금지 문구 활용**: "절대 누르지 마세요", "다운받지 마세요", "시간 낭비입니다" → 호기심 자극
- **별점 1점 = 성공**: 부정적 리뷰를 의도적으로 마케팅 소재로 활용
- **배신 인증 챌린지**: 사용자가 배신당한 순간을 캡처/녹화하여 SNS에 공유하도록 유도

### SNS 공유 유도 장치
- 서비스 내 **"배신 결과 공유하기"** 버튼 (개발팀 연계 필요)
- 공유 시 자동 생성 문구: "나는 [노래 제목]을 틀었는데 [배신 유형]을 당했습니다 #딴짓뮤직"
- **묵음챌린지**: 3번 연속 묵음 당하면 "축하합니다! 당신은 진정한 I'M KING" 뱃지 노출

### 타겟별 맞춤 메시지

| 타겟 | 핵심 메시지 | 주요 채널 |
|------|-------------|-----------|
| 밈 문화 애호가 | "세상에서 가장 쓸데없는 앱 등장" | 인스타 릴스, 유튜브 쇼츠 |
